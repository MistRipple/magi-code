# Magi 1.0.3 版本升级说明

发布日期：2026-03-14

## 重点更新
- Worker 侧只读工具引入“参数指纹去重”，同一工具且参数完全一致时短路执行，减少无效重复调用。
- 编排消息链路补齐 scope 相关字段，任务卡/指令卡的标识更稳定，状态回写更可靠。
- Worker 卡片与任务汇总链路进一步收敛，减少重复卡片与状态不一致问题。
- 设置面板交互与布局细节优化（开关与标题对齐、操作按钮一致性）。

## 修复与改进
- 修正部分任务卡片状态同步与完成态回写的时序问题。
- 强化工具结果处理与去重提示，降低只读工具循环触发概率。
- 清理并补全部分配置与模型能力注册逻辑的边界处理。

## 兼容性
- 保持现有配置与工具协议兼容，无需额外迁移操作。
