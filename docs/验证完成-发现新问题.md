## 📊 问题详情

### 1. TaskManager 双重使用

**统计**:
- TaskManager 使用: 10 处
- UnifiedTaskManager 使用: 17 处
- 两者职责高度重叠

**示例**:
```typescript
// 双重调用
this.taskManager?.updateSubTaskStatus(taskId, task.id, 'running');
this.unifiedTaskManager.startSubTask(mappedTaskId, task.id);
```

**影响**:
- 性能: 双重操作，重复持久化
- 维护: 需要同时维护两套代码
- 风险: 状态可能不一致

### 2. 状态同步代码

**发现位置**:
```typescript
// src/orchestrator/orchestrator-agent.ts:2357-2390
this.unifiedTaskManager.on('subtask:started', (task, subTask) => {
  // 同步到 TaskManager (old)
  if (this.taskManager) {
    this.taskManager.updateSubTaskStatus(task.id, subTask.id, 'running');
  }
});
```

**问题**: 这和之前的 TaskStateManager 问题完全一样！
