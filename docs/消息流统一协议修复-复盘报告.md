## 🏗️ 最终架构

### 消息流（统一版）
```
┌─────────────────────────────────────────────────────────┐
│                    CLI Adapter Layer                     │
│  (ClaudeAdapter, CodexAdapter, GeminiAdapter)           │
│  emit('output', chunk)                                   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│                   Normalizer Layer                       │
│  - ClaudeNormalizer, CodexNormalizer, GeminiNormalizer │
│  - parseChunk() + parseContentToBlocks()               │
│  - 输出：StandardMessage { blocks: ContentBlock[] }    │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│                 CLIAdapterFactory                        │
│  emit('standardMessage')                                │
│  emit('standardUpdate')                                 │
│  emit('standardComplete')                               │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│                  WebviewProvider                         │
│  postMessage({ type: 'standardMessage', ... })         │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│                    Webview UI                            │
│  - 接收 StandardMessage                                 │
│  - 渲染 blocks（文本/代码/工具调用/diff）              │
│  - 无需二次解析                                         │
└─────────────────────────────────────────────────────────┘
```

### 关键特性
1. **单一真相来源**：StandardMessage 是唯一的消息格式
2. **后端解析**：所有内容解析在后端完成（parseContentToBlocks）
3. **类型安全**：完整的 TypeScript 类型支持
4. **无重复通道**：删除了所有旧的事件通道（streamStart, cli:output, question）
5. **可测试性**：端到端测试覆盖核心流程
